# User Login System

## Pre-requisites
* XAMPP || LAMP || MAMP || any compatible server
* MySQL

## Instructions
1. Run the `db.sql` file in MySQL.
- Configure your MySQL login credentials in `/includes/db.php`
  
2. Login as admin with:
  - **Username:** `lain@root.sel`
  - **Password:** any value

3. get bored!
